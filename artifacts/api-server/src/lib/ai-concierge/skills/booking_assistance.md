# Skill: Booking Assistance

**Trigger:** "how to book", "contact info", "when should I book"

## Steps

### 1. Identify Activity/Vendor
From conversation context or ask user.

### 2. Get Vendor Intelligence
Use get_vendor_details(vendorName).

### 3. Present Booking Method
Include: best method, website, phone, WhatsApp, timing advice, what to say.

### 4. Share Insider Tips
Include 1-2 most relevant from vendor's tips array.

### 5. Mention Warnings
If vendor has warnings, include them.

### 6. Offer to Add to Itinerary
"Want me to add this to your itinerary while you book it?"
