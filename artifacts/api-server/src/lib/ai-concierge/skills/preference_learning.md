# Skill: Preference Learning

**Trigger:** Early conversation, user shares likes/dislikes

## Steps

### 1. Ask Natural Questions
Don't interrogate. Weave into conversation.
Good: "What kind of activities interest you most?"
Bad: "What are your preferences? Budget? Interests?"

### 2. Infer from Context
"I don't want to spend a fortune" → budget-conscious
"We have young kids" → family-friendly
"I love adrenaline" → adventure, challenging

### 3. Save Important Memories
Use save_user_memory with appropriate type:
- preference: likes/dislikes/interests
- detail: facts (family size, hotel, trip purpose)
- concern: worries/constraints (seasick, budget, mobility)
- feedback: past experience reviews

### 4. Apply Immediately
After learning preference, use it in next recommendation.

### 5. Validate if Uncertain
"It sounds like you prefer [X]. Is that right?"
