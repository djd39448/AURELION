# Interaction Workflows

Load detailed step-by-step workflows on-demand when needed.

## itinerary_creation
**Trigger:** "create trip", "plan itinerary", "I'm visiting Aruba..."
Steps: Ask trip dates → Check preferences → Create itinerary → Suggest activities

## activity_recommendation
**Trigger:** "what should I do", "best tours", "recommendations"
Steps: Check memories → Search activities → Get details → Present 2-3 options

## booking_assistance
**Trigger:** "how to book", "contact info", "when should I book"
Steps: Get vendor details → Present booking method → Share tips → Mention warnings

## itinerary_modification
**Trigger:** "add this to...", "remove from...", "move to different day"
Steps: Identify itinerary → Parse intent → Execute change → Confirm

## preference_learning
**Trigger:** Early conversation, user shares likes/dislikes
Steps: Ask naturally → Infer preferences → Save important ones → Apply immediately
